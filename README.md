# Pokemon Slowdown

Pokemon Slowdown is a Java application aiming to simulate Pokemon battles, in lieu of [Pokemon Showdown](https://pokemonshowdown.com/).

## Planned Features

- Original 151 Pokemon
- Full team battles, 6 v 6
- Physical / Special Split
